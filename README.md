# Kate's Crochet Corner 🧶

Welcome to the repository for **Kate's Crochet Corner**! This is a simple, responsive frontend web project I built to practice my web development skills. It serves as a landing page for a fictional crochet store, displaying various products and a newsletter signup form.

## 🚀 Project Overview

I created this project to get more hands-on experience with frontend frameworks and responsive design. The page features a product grid that adjusts seamlessly to different screen sizes, complete with interactive hover effects and a clean footer.

### ✨ Features
* **Responsive Product Grid:** Displays items using Bootstrap's grid system, ensuring the store looks great on desktop, tablet, and mobile.
* **Interactive UI:** Custom CSS was used to add smooth hover animations (scaling) to the product images.
* **Newsletter Footer:** A responsive subscription form styled with Bootstrap for users to join a mailing list.

## 🛠️ Built With

* **HTML5:** For the core structure and content.
* **CSS3:** For custom styling and hover animations.
* **Bootstrap 5:** Used via CDN for rapid UI development, grid layout, and pre-styled components (buttons, forms, typography).
